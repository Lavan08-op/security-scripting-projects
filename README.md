This repository contains my Cyber Security track. Each project is in its own folder
with source code, a dedicated README, and supporting documentation.

## Projects

### Project 1 — Password Strength Checker
A Python tool that evaluates password strength (Weak/Medium/Strong)
based on length, character variety, leaked-password blocklist checks,
and repeated-character pattern detection.
- Folder: `Project1-PasswordChecker/`
- Run: `python3 Project1-PasswordChecker/password_checker.py`
- Screenshot: `Project1-PasswordChecker/screenshots/`

### Project 2 — Caesar Cipher Tool
A menu-driven encryption/decryption tool implementing the Caesar
cipher, with input validation, a logging system, file save support,
and a brute-force demo illustrating why small key spaces are insecure.
- Folder: `Project2-CaesarCipher/`
- Run: `python3 Project2-CaesarCipher/caesar_cipher.py`
- Screenshot: `Project2-CaesarCipher/screenshots/`

### Project 3 — Phishing Awareness Analysis
Analysis of 3 sample phishing messages with identified red flags, a
non-expert triage checklist, and a decision tree for classifying
incoming threats (Safe / Suspicious / Malicious). Includes a
supporting Python script that scans message text for common
phishing red-flag keywords.
- Folder: `Project3-PhishingAnalysis/`
- Files: `phishing_analysis.md`, `phishing_scanner.py`
- Run: `python3 Project3-PhishingAnalysis/phishing_scanner.py`
- Screenshot: `Project3-PhishingAnalysis/screenshots/`
