# Project 3 - Phishing Awareness Analysis
DecodeLabs Internship - Week 3

Analysis of 3 sample phishing messages with red flags identified,
a non-expert triage checklist, and a decision tree for classifying
threats. Includes a Python script that scans pasted message text
for common red-flag keywords as a supporting tool.

## Files
- phishing_analysis.md - full writeup with samples, checklist, decision tree
- phishing_scanner.py - keyword-based red flag scanner (run: python3 phishing_scanner.py)
